Simple plugin to show an image, photo or logo in a widget with text and link

== Description ==

Cresta Image In Widget it's a widget that allows you to simply add an <strong>image in a widget</strong>, choose the size, choose the border radius, add a link on it and add text under the image.

<strong>Some features</strong>
<ul>
	<li>Add an image in the widget</li>
	<li>Choose the image size</li>
	<li>Add a border radius at the image</li>
	<li>Choose where to show the widget (entire website, only home page, only all posts, etc...)</li>
	<li>Add a link on the image</li>
	<li>Add text under the image</li>
</ul>

This plugin can be useful if you want to show an "About Me" section in your sidebar, or the logo of your company in your footer, or for any reason you need an image in the widget.
All features are optional so you can choose what to show in your website and what not!

<p>
</p>

== Installation ==

1. Unzip cresta-image-in-widget.zip
2. Upload the folder 'cresta-image-in-widget' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress Dashboard
4. Go to WordPress Main Menu -> Appearance-> Widgets and choose the "Cresta Image In Widget" widget


== Frequently Asked Questions ==

= Can I use several times the widget? =
Yes, the widget can be used 1 or more times in the same page.

= Can I choose where to show the widget? =
Yes, in the widget settings you can choose where to show the widget (Entire website, Only front page, Only blog page, Only all posts, Only all pages, Only all posts and pages).

= Can I change the text color or add effects to the image? =
No, bacause the widget uses the theme style you are using.
